# prefixTuning-Dialogue-State-Tracking-Module
Apply prefix tuning to adapting encoder-decoder architecture for Dialogue State Tracking Module implementation

### Dataset
In this project, we used the dataset MultiWOZ 2.2 and WOZ 2.0. The preprocessed dataset can be downloaded [here](https://github.com/PaddlePaddle/Knover/tree/develop/projects/AG-DST#Preparation).
